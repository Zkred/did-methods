# @zkred/did-core

Shared types and utilities for the [`@zkred` DID method implementations](https://github.com/Zkred/did-methods):
W3C DID Core data-model types, standard resolution error codes, and an HTTP JSON fetch helper
that maps failures onto DID resolution errors.

You normally don't install this directly — it comes in as a dependency of
`@zkred/did-webplus` or `@zkred/did-webvh`.
